## 🛠️ Tecnologias

- [Reveal.js 5.1](https://revealjs.com/) — framework de apresentações
- [Lucide Icons](https://lucide.dev/) — ícones SVG
- CSS customizado com tema dark/cosmos
- Efeitos de partículas e starfield em JavaScript

## ▶️ Como Usar

### Opção 1 — Abrir direto no navegador
```
Abra o arquivo index.html no seu navegador
```

### Opção 2 — Servidor local
```bash
# Com Python
python -m http.server 8000

# Com Node.js
npx serve .
```

Depois acesse `http://localhost:8000`

### Navegação
- **→** / **↓** — próximo slide
- **←** / **↑** — slide anterior
- **ESC** — visão geral dos slides
- **F** — tela cheia
- Clique nos cards do sumário para navegar diretamente
